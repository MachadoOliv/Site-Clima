🌍 Weather Forecast App

A responsive and dynamic web application that allows users to check the current weather conditions for any city around the world in real-time.

🚀 Features

* **Global Search:** Find weather details for any location by simply typing the city name.
* **Comprehensive Data:** Displays current temperature, minimum and maximum temperatures, humidity, wind speed, and overall sky conditions (e.g., clear, cloudy, rainy).
* **Dynamic Feedback:** Instant data fetching from a public weather API.

🛠️Technologies Used

*Frontend:** HTML5, CSS3, JavaScript (ES6+)
*API:** Public Weather API (REST)
*Development Approach:** Built efficiently within a **3-day timeline**, leveraging AI assistance to optimize workflow, automate boilerplate tasks, and speed up delivery.

📄 How to Use

1. Enter the name of the city in the search bar.
2. Click the search button or press Enter.
3. View the detailed, real-time weather metrics instantly.
